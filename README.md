# Robustness

## Installation

* Clone the repo and cd into it:
    ```bash
    git clone https://github.com/danielmamay/robustness.git
    cd robustness
    ```
* Use Python 3.9 or later:
    ```bash
    conda create -n robustness python=3.9
    conda activate robustness
    pip install -r requirements.txt
    ```
